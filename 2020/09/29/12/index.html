    Ubuntu + Metasploit + CobaltStrike4.0

    内网渗透利器msf和cs是必不可少的，这两大利器的联动能达到更好的效果。所以就先从搭建属于自己的内网渗透平台开始做起。

    Ubuntu 搭建

    Ubuntu 18.04 http://mirrors.aliyun.com/ubuntu-releases/18.04/
    之后在VMware中创建之后启动起来，设置用户名密码完成。配置一下以下几点平时使用更方便：

    Ⅰ 使用root登陆

    sudo passwd root
设置密码
重复密码

    使用root进行登录

    Ⅱ 安装SSH服务

    sudo apt install openssh-server
sudo service ssh start
sudo apt install openssh-client

    Ⅲ SSH开启ROOT登陆

    vim /etc/ssh/sshd_config
添加或修改 PermitRootLogin yes

    最后SSH使用ROOT登陆

    Metasploit 搭建

    在根目录创建msf文件进入后执行

    curl https://raw.githubusercontent.com/rapid7/metasploit-omnibus/master/config/templates/metasploit-framework-wrappers/msfupdate.erb > msfinstall && chmod 755 msfinstall && ./msfinstall

    安装后msfconsole进入

    简单无脑，直接进入。

    CobaltStrike4.0 搭建

    将CobaltStrike4.0.zip文件夹复制到/cs4/下后执行

    unzip CobaltStrike4.0.zip
cd CobaltStrike4.0/
./teamserver 192.168.27.47 wlsmile123..

# ./teamserver 本地IP 密码

    在本地windows打cs客户端，输入用户名密码。

    连接。
